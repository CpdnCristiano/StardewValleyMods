# Full Inventory View

A quality-of-life mod for Stardew Valley.

This mod lets you view all of your inventory rows at once when opening menus, removing the need to scroll.

## Features
*   Expands the inventory display grid so all unlocked rows (up to the maximum capacity) are visible simultaneously.
*   Compatible with most vanilla and modded inventory UI layouts.

## Installation
1. Install the latest version of [SMAPI](https://smapi.io).
2. Download this mod and extract it into your `Stardew Valley/Mods` folder.
3. Run the game using SMAPI.
