# Fix Warp Greenhouses

A compatibility mod for Stardew Valley.

This mod corrects warp and teleportation logic when using multiple greenhouses or greenhouse expansion mods, ensuring you land at the proper exit and entry coordinates.

## Features
*   Fixes exit warps from custom greenhouses back to the farm map.
*   Resolves coordinates mismatch when multiple greenhouses are active.

## Installation
1. Install the latest version of [SMAPI](https://smapi.io).
2. Download this mod and extract it into your `Stardew Valley/Mods` folder.
3. Run the game using SMAPI.
