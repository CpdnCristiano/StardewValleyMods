[b][size=5]Fix Warp Greenhouses[/size][/b]

[b]Description[/b]
Fix Warp Greenhouses is a mod for Stardew Valley that fixes the teleport and warp logic when using multiple greenhouses, as well as complex custom greenhouses with seasonal sub-locations (interiors) or basements. 

In the base game, if you have more than one greenhouse (e.g., via Build More Greenhouses), you always exit from the default/main greenhouse when using an internal warp. This mod intercepts the warp target resolution, ensuring that when you enter a specific greenhouse building, you always return to and exit from that exact building.

Additionally, this mod features a [b]dynamic physical cloning system[/b]. When using custom greenhouses with interior sub-locations (like basements, cellars, or seasonal variations), the mod automatically duplicates these interior maps uniquely for each greenhouse you construct. This ensures that your crops, chests, casks, and decorations are kept separate and persistent.

[b]Main Features[/b]
✅ Fixes internal teleportation and exit warps between multiple greenhouses.
✅ Automatically clones and isolates interior sub-locations (cellars, basements, seasonal maps) for each constructed greenhouse.
✅ Ensures consistency when entering and exiting different greenhouses.
✅ Solves the teleport issue in [b][url=https://www.nexusmods.com/stardewvalley/mods/3969]Oasis Greenhouse[/url][/b], keeping its basement warp separate.
✅ Compatible with mods that add multiple greenhouses, such as [b][url=https://www.nexusmods.com/stardewvalley/mods/21778]Build More Greenhouses[/url][/b].

[b]How It Works[/b]
The mod scans the warp graph of all greenhouses in your farm, detects their interior sub-locations, and dynamically clones them for each physical greenhouse building. The mod redirects warps and map assets on the fly so everything maps correctly to the original graphic files, avoiding missing texture errors.
